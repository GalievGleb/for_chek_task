Неизменяемые типы данных:

Строки (str)
Пример: text = "Привет"
Нельзя изменить одну букву: text[0] = "П" вызовет ошибку.
Если нужно изменить строку, создаем новую: text = text + "!"

Числа (int, float)
Пример: number = 42
Изменить number напрямую нельзя, но можно создать новое число: number = number + 1

Кортежи (tuple)
Пример: data = (1, 2, 3)
Нельзя заменить элемент: data[0] = 10 вызовет ошибку.
Если нужно изменить, создаем новый кортеж: data = (10,) + data[1:]

Если данные неизменяемые, Python работает с ними быстрее. Неизменяемые объекты можно использовать в качестве ключей словаря.
Когда нужно хранить изменяемые данные, используем списки (list) или словари (dict).
